📖 다운로드한 DOCX 파일을 한글(HWP)에서 여는 방법

워드(DOCX) 파일은 한글 2014 이상에서 바로 열 수 있습니다.

  1. DOCX 파일 다운로드 — 위 버튼으로 파일을 받으세요
  2. 한글 프로그램 실행 — 한컴오피스 한글을 엽니다
  3. 파일 → 불러오기 — 메뉴에서 [불러오기] 선택
    한글 메뉴: 파일(F) → 불러오기(O) 또는 단축키 Ctrl+O
    파일 형식을 "마이크로소프트 워드 문서(*.docx)"로 선택
  4. DOCX 파일 선택 — 다운로드한 소방계획서 파일을 선택하고 [열기]
  5. 내용 확인 후 HWP로 저장 — 내용을 확인·수정한 뒤
    파일(F) → 다른 이름으로 저장(A)에서 파일 형식을 HWP로 저장하면 완료!

💡 팁: 한글에서 열면 표 서식이 자동 변환됩니다. 기존 소방계획서 HWP 양식에 내용을 복사-붙여넣기 하셔도 됩니다.

📋 팁: 한글 2010 이하 버전을 사용하시는 경우, 텍스트 다운로드 버튼으로 받은 텍스트를 복사하여 기존 HWP 서식에 직접 붙여넣기 하세요.
